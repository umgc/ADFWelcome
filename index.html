How to Git

GIT is an open-source version control system. An interactive tutorial is available at https://learngitbranching.js.org/ .

Your workflow will probably look something like this:

  1. Start the Android emulator and Visual Studio Code
  2. Open a new terminal, and clone your repository:
    git clone https://github.com/umgc/DevSecOpsPlayground.git
  3. Create a new branch:
    git checkout -b jersoe/buttonfix
  4. Make changes, and test locally
  5. Stage your changes using:
    git add .
  6. Commit your changes using:
    git commit -m "Finally got that button to work"
  7. Push your changes to GitHub:
    git push --set-upstream origin jersoe/buttonfix
  8. Create a pull request on GitHub
  9. Wait until your PR is approved
  10. Merge your PR on GitHub into the appropiate branch (ie. developer or main)

How to use the terminal

The terminal can be used to interact with the system using the command line interface. You can issue commands and see the output. To open a terminal, use the terminal button at the bottom of the screen or from within VS Code.

Here are some commands that are used often:

  • Navigate to a directory: cd <directory>
  • Move a file: mv <file> <destination>
  • Rename a file: mv <file> <destination>
  • Copy a file: cp <file> <destination>
  • Delete a file: rm <file>
  • Delete a directory: rm -rf <directory>
  • Execute a command as root: sudo <command>
  • Become root: sudo su
  • Untar a file: tar -xvf <tar_archive>
